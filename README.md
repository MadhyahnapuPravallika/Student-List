Student-List:It demonstrates the use of ng-repeat, filters, and custom directives to manage a dynamic student list.
Users can dynamically add students, search by name, and automatically sort data while highlighting the top performer.

Features:
Add Students Dynamically – Enter Name, Marks, and Grade
Search Functionality – Filter students by name in real time.
Automatic Sorting – Students are sorted alphabetically by name and numerically by marks.
Highlight Top Performer – A custom AngularJS directive highlights the student with the highest marks.
Responsive UI – Clean layout using Bootstrap and custom CSS.

Concepts Used:
ng-app – Defines the AngularJS application.
ng-controller – Connects HTML view with the controller logic.
ng-model – Creates two-way data binding between inputs and variables.
ng-repeat – Dynamically renders student data from the array.
filter – Filters the student list based on the search text.
orderBy – Sorts the student data automatically.
Custom Directive (highlightTopper) – Highlights the student with the highest marks.

How to Run:
Clone or download the repository.
Open the folder in VS Code or any code editor.
Open index.html directly in the browser or run it via Live Server.
Add students dynamically and test search, sorting, and highlight features.
